# Quick Start

Welcome to Devify Documentation, your gateway to a versatile and hassle-free testing and development environment for your applications.

## How to Use It

Devify is your go-to solution for creating, testing, and refining your applications. Designed as a Fake API, it's tailored to meet the needs of developers like you who seek a reliable testing and development environment. Whether you're building a new web or mobile application, or fine-tuning an existing one, Devify has got you covered.

In this section we will talking about how to start using Devify. It's very simple you can use tools like [Postman](https://www.postman.com) to geting data from Devify or [cURL](https://curl.se).

For example if your use [cURL](https://curl.se), you can run the following command in your command line:

```bash
$ curl -o C:\output.json -X GET "https://api-devify.my.id/v1/products"
```

OR

```bash
$ curl -X GET "https://api-devify.my.id/v1/products"
```

## Examples Codes

Devify, a fictitious API, is a playground designed to offer developers a safe and versatile testing environment for their applications. It serves as a valuable resource for developers to experiment with various HTTP methods and fine-tune their coding skills.

Devify supports a wide range of HTTP methods, allowing developers to gain hands-on experience in working with APIs. Here are some examples of how to use these methods in your development projects:

---

### Retrieve Data

Whether you're working on web applications, mobile apps, or any other software project, Devify simplifies the process of acquiring dummy data.

You can make GET requests to obtain the data they need, simulating various scenarios and data structures. This enables thorough testing, debugging, and the creation of reliable code.

```js
const baseUrl = "https://api-devify.my.id";

async function products() {
  try {
    const response = await fetch(`${baseUrl}/v1/products?limit=3`);
    const result = await response.json();

    return result;
  } catch (err) {

  }

products().then((datas) => console.log(datas));
```

OR


### Create Data

Devify allows you to customize your data to fit the specific requirements of your application. You can define data structures, attributes, and relationships, tailoring the generated data to match your exact needs.

The quick data creation process accelerates your development cycle. This agility lets you focus more on coding and refining your application, reducing the time needed for data setup.

### Update Data

Devify's straightforward and intuitive API endpoints make it easy to implement product updates into your development workflow. By using the provided endpoints, you can simulate real-world scenarios, ensuring that your application can handle changes in product data efficiently.

With Devify, you can modify a product's attributes, such as its name, description, price, and availability, allowing you to test your application's handling of product updates in a safe and controlled environment.

This is a valuable resource for developers who want to ensure their applications are responsive to changes in product information while avoiding any potential mishaps that could occur in a live system.

### Update Product By ID

```js
const baseUrl = "https://api-devify.my.id";

let content = new FormData();
content.append("image", file);
content.append("title", "This is my title. (updated)");
content.append("description", "This is my description. (updated)");
content.append("price", 15);
content.append("category", "mens");

async function updateProduct() {
  try {
    const res = await fetch(`${baseUrl}/v1/products/1`, {
      method: "PATCH",
      body: content,
    });
    const result = await res.json();

    return result;
  } catch (err) {
    return null;
  }
}

updateProduct().then((result) => console.log(result));
```

### Remove Data

To remove data using Devify, you can make HTTP requests to the specific API endpoints designed for data deletion. These endpoints are typically accessible through HTTP DELETE requests.

You can specify the resource or data you want to remove by providing the necessary parameters, such as IDs or unique keys, depending on your API structure.

For example, if you want to delete a specific product, you can send an HTTP DELETE request to the "v1/products/1" or "v1/products/key/650698cfb8318065fca250ef" endpoint. Devify will handle the data removal process, allowing you to keep your testing and development environment tidy and up to date.

### Delete Product By ID

```js
const baseUrl = "https://api-devify.my.id";

async function deleteProduct() {
  try {
    const res = await fetch(`${baseUrl}/v1/products/1`, { method: "DELETE" });
    const result = await res.json();

    return result;
  } catch (err) {
    return null;
  }
}
```
